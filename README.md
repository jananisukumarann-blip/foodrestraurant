
## 🍽️ Restaurant Website

This is a modern and responsive restaurant website designed to showcase delicious cuisine, signature dishes, and an elegant dining experience. 
The website provides a visually appealing layout with smooth navigation, making it easy for users to explore the menu, learn about the restaurant, and get in touch.

---

## 🚀 Features

* Responsive design for all devices (mobile, tablet, desktop)
* Attractive menu and food showcase sections
* Clean and modern UI
* Smooth navigation and structured layout
* User-friendly design focused on customer experience

---

## 🛠️ Technologies Used

* **HTML5** – Structure and content
* **CSS3** – Styling, layout, and responsiveness
* **JavaScript** – Interactivity and dynamic behavior
* **Bootstrap** – Responsive components and grid system

---

## 📌 Purpose

This project was built to practice front-end web development skills and to create a professional restaurant website that highlights food, ambiance, and brand identity.
